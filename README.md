##  Featured Projects

### **Predicting Salmon Returns with Machine Learning**
**Forecasting salmon abundance across three river systems using ML models.**  
- Built predictive time-series models for environmental decision support  
- Integrated salmon return records (age classes, total spawners) with large-scale oceanographic indices
- [Project Website](https://salmonprize.com/), [Code](https://github.com/schmidj/PredictSalmonRuns)

### **Predicting Recreational Angler Behavior from App data**
**Spatiotemporal ML models using thousands of app records.**  
- Modeled recreational fishing activity over time and space  
- Applied a broad set of ML models including GLMs, random forests, SVMs, and neural networks
- [Schmid et al. (2025)](https://arxiv.org/abs/2409.17425), [Code](https://github.com/schmidj/PredictingAnglerActivityOntario)  <br>
